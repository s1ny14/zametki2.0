Модули проекта

Главный модуль приложения «Менеджер заметок».

Этот модуль запускает графическое приложение для управления заметками. Приложение позволяет создавать, просматривать, редактировать и удалять заметки с поддержкой тегов, приоритетов и статусов.

Пакет notebook

Пакет notebook - модуль для управления заметками.

Этот пакет предоставляет функциональность для создания, хранения и управления заметками с поддержкой тегов и метаданных.

Modules:

models: Определение класса Note и методов работы с заметками storage: Класс для сохранения и загрузки заметок из JSON-файла

Classes:
